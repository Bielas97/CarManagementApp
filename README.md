# Car Management App

This project was built for personal purposes to show abilities in Java 1.9 modules and in Java 1.8 streams
## Description

The project helps managing cars

## Detailed Description

User provides .json files filled with object of a Car, the app is reading it and provides the user to do the 
following operations:

1. show all cars
2. find cars by carBody between price
3. find car by engine type
4. show statistics
5. show mileage for each car
6. show cars of tyre type
7. sort cars by field
7. show cars with given components

all stream operations take place in service.src.main.java.com.service.CarStore
